# German Anki Preparation — Специфікація

## Огляд

AI Artifact (Claude-powered) що приймає сирий список слів (німецьких або українських, змішаних) і повертає відформатований CSV для імпорту в Anki. Обробка відбувається у два етапи: автоматична нормалізація → черга уточнень для неоднозначних випадків.

---

## Вхідні дані

- Одне слово або фраза на рядок
- Мова: німецька, українська або змішано в одному списку
- Опціональна підказка через `=`: `hund = собака`, `See = озеро`
- Якщо артикль вказано явно (`der See`) — це однозначне слово, питань немає

---

## Вихідні дані — формат CSV

Два поля, роздільник **табуляція** (`\t`). Крапка з комою використовується всередині полів (кілька значень), тому як роздільник колонок не підходить.

```
<поле 1 — DE з граматикою><TAB><поле 2 — UA переклад>
```

Приклади:
```
der Hund, -es, ¨-e	собака
fahren (fährt, fuhr, ist gefahren)	їхати
stolz auf +Akk (stolzer, am stolzesten)	гордий (кимось/чимось)
weil [sub]	тому що [дієслово — в кінець речення]
```

Кілька перекладів (всередині другого поля):
- Близькі за змістом → через кому: `гарний, красивий`
- Різні значення → через крапку з комою: `том (книги); стрічка; гурт`

## Граматичні шаблони по частинах мови

### Nomen

```
der Hund, -es, ¨-e
die Frau, -, -en
das Auto, -s, -s
der Mensch, -en, -en
der Vater, -s, ¨
die Leute (pl), -, -
```

Три позиції: `артикль+слово, Genitiv-закінчення, Plural-закінчення`

Правила:
- `-` = без змін або форма відсутня
- `¨` = умлаут (без додаткового закінчення)
- `¨-e` = умлаут + закінчення
- `(pl)` = слово існує тільки в множині — ставити після слова, Genitiv і Plural: `-`
- n-Deklination: обидва поля `-en`

### Verb

```
kaufen (kaufte, hat gekauft)
fahren (fährt, fuhr, ist gefahren)
lesen (liest, las, hat gelesen)
an|rufen (rief an, hat angerufen)
sich freuen (freute sich, hat sich gefreut)
warten (wartete, hat gewartet); auf +Akk
sich erinnern (erinnerte sich, hat sich erinnert); an +Akk
```

Структура дужок: `(3sg-Präsens якщо є Vokalwechsel, Präteritum, hat/ist Partizip II)`

Правила:
- 3sg у дужках **тільки якщо є зміна голосної** (fahren→fährt, lesen→liest)
- Розділюване дієслово: `|` між префіксом та коренем
- Reflexiv: `sich` перед інфінітивом
- Rektion: після крапки з комою: `; Präp +Kasus`
- `hat` скорочується як є, `ist` як є (не скорочувати)

### Adjektiv

```
schnell (schneller, am schnellsten)
gut (besser, am besten)
hoch (höher, am höchsten)
stolz auf +Akk (stolzer, am stolzesten)
müde (müder, am müdesten)
```

Структура: `слово Rektion-якщо-є (Komparativ, Superlativ)`

Правила:
- Компаратив показувати **для всіх** прикметників
- Rektion перед дужками: `stolz auf +Akk (...)`
- Переклад прикметника з Rektion включає підказку: `гордий (кимось/чимось)`

### Adverb

```
leider
gern (lieber, am liebsten)
bald (eher, am ehesten)
```

Правила:
- Нерегулярний компаратив: показувати в дужках (як прикметник)
- Регулярний або без компаративу: просто слово
- Rektion у прислівників немає

### Präposition

```
mit +Dat
durch +Akk
wegen +Gen
in +Dat/+Akk [wo/wohin]
an +Dat/+Akk [wo/wohin]
```

Правила:
- Wechselpräpositionen (in, an, auf, über, unter, vor, hinter, neben, zwischen): `+Dat/+Akk [wo/wohin]`

### Konjunktion

```
und [koord]
weil [sub]
deshalb [adv]
entweder...oder [koord]
```

Типи:
- `[koord]` — координуючий: порядок слів не змінюється
- `[sub]` — підрядний: дієслово йде в кінець підрядного речення
- `[adv]` — прислівниковий: дієслово на 2 місці (інверсія підмета)

Переклад включає граматичну підказку: `тому що [дієслово — в кінець]`

### Partikel (Modalpartikeln)

```
doch [partikel]<br>Das ist doch klar!
mal [partikel]<br>Komm mal her.
```

Структура: `слово [partikel]<br>приклад речення`

Переклад: пронумеровані значення з мікроприкладом:
```
1) підкреслення очевидного: "Та це ж ясно!"
2) заперечення у відповідь: Doch! = "Та ні!"
```

### Pronomen

```
jemand
niemand
etwas
```

Просто слово, без граматики.

### Zahlwort

```
zwanzig
hundert
```

Просто слово, без граматики.

### Phrase (стала фраза)

```
auf jeden Fall
in der Regel
```

Просто фраза, без граматичних міток.

### Redewendung (ідіома)

```
jdm. auf die Nerven gehen [Rdw]
etw. in Kauf nehmen [Rdw]
```

Мітка `[Rdw]` в кінці. `jdm.` = jemandem, `etw.` = etwas.

## Легенда символів

| Символ | Значення |
|--------|----------|
| `¨` | умлаут у формі |
| `¨-e` | умлаут + закінчення -e |
| `-` | без змін / форма відсутня |
| `(pl)` | слово існує тільки в множині |
| `\|` | межа розділюваного дієслова (an\|rufen) |
| `sich` | зворотне дієслово |
| `+Akk` / `+Dat` / `+Gen` | відмінок після прийменника |
| `+Dat/+Akk` | Wechselpräposition |
| `[wo/wohin]` | підказка для Wechselpräposition |
| `[koord]` | координуючий сполучник |
| `[sub]` | підрядний сполучник |
| `[adv]` | прислівниковий сполучник |
| `[partikel]` | модальна частка |
| `[Rdw]` | Redewendung (ідіома) |
| `[ugs.]` | розмовне слово |
| `jdm.` | jemandem (комусь, Dat) |
| `etw.` | etwas (щось, Akk) |
| `<br>` | перенос рядка в Anki-картці |

## Обробка — пайплайн

```
Вхідний список
    ↓
1. Детекція мови (DE / UA / мішано)
    ↓
2. Нормалізація (інфінітив, однина де можливо)
    ↓
3. Видалення дублікатів (мовчки)
    ↓
4. Визначення PoS і граматичних форм
    ↓
5. Перевірка на неоднозначності → черга уточнень
    ↓
6. Генерація CSV для готових слів
    ↓
7. Сортування за частотою вживання
    ↓
8. Вивід: [CSV блок] + [блок уточнень якщо є]
    ↓
9. Після відповідей → доповнення CSV → повторне сортування
```

### Сортування за частотою

Слова сортуються за частотою вживання в сучасній німецькій мові (корпусна частота). Найчастотніші — першими, рідкісні — в кінці. Мета: Anki показує важливіші слова раніше.

Правила сортування:
- Базова форма (інфінітив / однина) використовується для пошуку частоти
- Українські слова сортуються за частотою їхнього **німецького відповідника**
- Фрази та ідіоми (`[Rdw]`) — завжди в кінці списку, після окремих слів
- Якщо частота невідома — слово додається в кінець перед ідіомами

## Обробка неоднозначностей і помилок

### Категорія 1 — Слово не знайдено

| Ситуація | Поведінка |
|----------|-----------|
| Очевидна опечатка | Запропонувати схожі слова: "Можливо: `fahren`, `fallen`?" |
| Можливий сленг | Уточнити: "Це може бути розмовне від `nichts` — правильно?" |
| Слово не знайдено і схожих немає | Позначити як не знайдено, вивести в блок уточнень |
| Власна назва | Запитати: "Це власна назва — додати як є чи пропустити?" |

### Категорія 2 — Артикль змінює значення

Якщо слово написано **без артикля** і існує з різними артиклями:

```
⚠️ "See" — кілька значень:
  [a] der See — озеро
  [b] die See — море
  [c] обидва (дві окремі картки)
```

Якщо артикль вказано явно → обробляти без питань.

### Категорія 3 — Форма слова не базова

| Форма | Поведінка |
|-------|-----------|
| Дієслово не в інфінітиві (`fuhr`) | Нормалізувати мовчки → `fahren` |
| Прикметник відмінений (`schnellen`) | Нормалізувати мовчки → `schnell` |
| Розділюване у спряженні (`ruft an`) | Нормалізувати мовчки → `an\|rufen` |
| Іменник у множині, однозначна форма (`Hunde`) | Нормалізувати мовчки → `der Hund` |
| Іменник у множині, неоднозначна форма (може бути від двох слів) | Вивести в уточнення з варіантами |
| Слово існує тільки в множині (`Leute`) | Залишити як є з міткою `(pl)` |

### Категорія 4 — Неоднозначна частина мови або значення

Тільки якщо неоднозначність **суттєва** (різна граматика або значення):

```
⚠️ "übersetzen" — два різних дієслова:
  [a] über|setzen (fuhr über, ist übergesetzt) — перевезти через
  [b] übersetzen (übersetzte, hat übersetzt) — перекладати
```

```
⚠️ "Band" — три різних слова:
  [a] der Band, -(e)s, ¨-e — том, книга
  [b] das Band, -(e)s, ¨-er — стрічка, зв'язок
  [c] die Band, -, -s — музичний гурт
  [d] усі три
```

Варіанти завжди супроводжуються **перекладом і коротким контекстом** українською.

### Категорія 5 — Підказка від користувача конфліктує з граматикою

| Ситуація | Поведінка |
|----------|-----------|
| Неправильний артикль (`die Hund`) | Виправити мовчки, повідомити: "Виправлено: `die` → `der`" |
| Підказка-переклад вказує на конкретне значення (`See = озеро`) | Використати `der See`, не питати |
| Підказка неточна але зрозуміла (`gut = гарний`) | Використати, додати інші значення в переклад |

### Категорія 6 — Українське слово на вході

| Ситуація | Поведінка |
|----------|-----------|
| Однозначний переклад (`собака`) | Автоматично → `der Hund, -es, ¨-e` |
| Кілька перекладів з різним контекстом (`іти`) | Запитати: "Яке значення?" з варіантами і поясненням |
| Абстрактне або багатозначне слово (`відчуття`, `стан`) | Запитати контекст використання, запропонувати варіанти |
| Кілька перекладів — користувач може вибрати кілька | Дозволити вибір `[a][b][c]` або `[усі]` |

### Категорія 7 — Фраза чи окремі слова

| Ситуація | Поведінка |
|----------|-----------|
| Відома стала фраза (`auf jeden Fall`) | Автоматично як одна картка |
| Відома ідіома (`jdm. auf die Nerven gehen`) | Автоматично з міткою `[Rdw]` |
| Невідома комбінація (`auf dem Tisch`) | Запитати: "Це стала фраза для вивчення чи окремі слова?" |

## Дублікати

- Видаляти мовчки
- Якщо одне слово з'являється в різних формах (однина + множина того самого іменника) → залишити одну нормалізовану форму
- Якщо одне написання може бути двома різними словами (різний артикль) → вивести в уточнення (Категорія 2)

---

## Формат виводу в Artifact

```
✅ ГОТОВО — 27 карток

[CSV-блок для копіювання]

────────────────────────
⚠️ ПОТРЕБУЄ УТОЧНЕННЯ — 3 пункти

1. "See" — кілька значень:
   [a] der See, -s, -n — озеро (стояча вода, inland)
   [b] die See, -, -n — море (відкрита вода)
   [c] обидва

2. "übersetzen" — два різних дієслова:
   [a] über|setzen — перевезти через річку/кордон
   [b] übersetzen — перекладати текст

3. "іти" — яке значення руху?
   [a] gehen — іти пішки
   [b] fahren — їхати транспортом
   [c] laufen — бігти або іти (розмовне)
   [d] усі три (три окремі картки)
```

Після відповідей — CSV доповнюється і виводиться повний фінальний блок із сортуванням.

Відповіді на уточнення — текстом (`a`, `b`, `c` або `усі`). Якщо питань багато, всі відповіді можна надати в одному повідомленні списком:
```
1. b
2. a
3. a, c
```

## Документація для користувача (вбудована в Artifact)

Artifact містить розкривний блок "Як читати картки" з легендою символів і прикладами для кожної частини мови. Опис типів сполучників ([koord]/[sub]/[adv]) включає приклад зміни порядку слів українською.
