Full decision trees for every ambiguity category.

## Категорія 1 — Слово не знайдено

```
Слово не знайдено
    ├── Є схожі слова (відстань редагування ≤ 2)?
    │     → Запропонувати у черзі: 
    │       '⚠️ "bleibn" не знайдено. Можливо: bleiben (залишатись)?'
    │       [a] Так, це bleiben
    │       [b] Ні, ось правильне слово: ___
    │
    ├── Схоже на сленг/скорочення?
    │     → '⚠️ "nix" — можливо розмовне від "nichts"?'
    │       [a] Так
    │       [b] Ні, інше: ___
    │
    ├── Схоже на власну назву (велика літера, місто, ім'я)?
    │     → 'Це власна назва — додати як є чи пропустити?'
    │       [a] Додати як є
    │       [b] Пропустити
    │
    └── Нічого не підходить
          → Позначити: '⚠️ "xyz" — слово не знайдено. Перевір написання.'
            Вивести в кінець черги уточнень.
```

## Категорія 2 — Артикль змінює значення

Відомі омоніми з різними артиклями (неповний список для довідки):

| Слово | Варіанти |
|-------|---------|
| See | der See (озеро) / die See (море) |
| Band | der Band (том) / das Band (стрічка) / die Band (гурт) |
| Leiter | der Leiter (керівник) / die Leiter (драбина) |
| Tau | der Tau (роса) / das Tau (канат) |
| Steuer | das Steuer (кермо) / die Steuer (податок) |
| Verdienst | der Verdienst (заробіток) / das Verdienst (заслуга) |
| Gehalt | der Gehalt (вміст) / das Gehalt (зарплата) |
| Schloss | das Schloss (замок-будівля / замок-засув) → одне слово, але два значення |
| Tor | das Tor (ворота/гол) / der Tor (дурень) |
| Golf | der Golf (затока/гольф) / das Golf (марка авто) |

**Правило:**
- Без артикля + слово є омонімом → черга уточнень з усіма варіантами
- З артиклем → обробляти без питань
- З підказкою `= переклад` → вибрати відповідний артикль, не питати

**Формат питання:**
```
⚠️ "Band" — кілька різних слів:
  [a] der Band, -(e)s, ¨-e — том, книга зібрання
  [b] das Band, -(e)s, ¨-er — стрічка, зв'язок
  [c] die Band, -, -s — музичний гурт
  [d] усі три (три окремі картки)
```

## Категорія 3 — Форма не базова

| Форма | Визначення | Дія |
|-------|-----------|-----|
| Дієслово (fuhr, ging, kaufte) | Präteritum або спряження | Нормалізувати мовчки |
| Дієслово (gefahren, gekauft) | Partizip II | Нормалізувати мовчки |
| Прикметник (schnellen, gutem) | Відмінена форма | Нормалізувати мовчки |
| Розділюване (ruft an, fuhr ab) | Спряжена форма | Нормалізувати мовчки |
| Іменник у множині — однозначно | Hunde → Hund | Нормалізувати мовчки |
| Іменник у множині — неоднозначно | Bänder → Band (das) або ??? | Черга уточнень |
| Тільки множина (Leute) | Pluralia tantum | Залишити, додати `(pl)` |

**Визначення неоднозначної множини:**
Якщо форма множини збігається з Nom.Sg або є формою від двох різних іменників → питати.

## Категорія 4 — Неоднозначна частина мови / значення

Питати тільки коли граматика **суттєво відрізняється**:

### Trennbar vs. nicht-trennbar (різне значення і граматика):
```
übersetzen:
  [a] über|setzen (fuhr über, ist übergesetzt) — перевезти через
  [b] übersetzen (übersetzte, hat übersetzt) — перекладати

umfahren:
  [a] um|fahren (fuhr um, hat umgefahren) — збити машиною
  [b] umfahren (umfuhr, hat umfahren) — об'їхати навколо

wiederholen:
  [a] wieder|holen (holte wieder, hat wiedergeholt) — принести назад
  [b] wiederholen (wiederholte, hat wiederholt) — повторювати
```

### Adj vs. Adverb (зазвичай не питати — форма однакова):
`schnell`, `gut`, `früh` — як adj і adv виглядають однаково, обробляти як adj.

### Verb vs. Nomen (якщо капіталізація неоднозначна):
`laufen` vs. `das Laufen` — якщо написано з малої → verb, з великої → Nomen.

## Категорія 5 — Підказка конфліктує

| Ситуація | Дія |
|----------|-----|
| Неправильний артикль (`die Hund`) | Виправити мовчки, повідомити: `✏️ Виправлено: die → der Hund` |
| Артикль не існує для цього роду | Виправити і повідомити |
| Підказка-переклад вказує на значення (`See = озеро`) | Використати `der See`, не питати |
| Підказка частково неточна (`gut = гарний`) | Використати, розширити переклад |
| Підказка суперечить граматиці | Повідомити і запитати підтвердження |

## Категорія 6 — Українське слово на вході

```
UA слово
    ├── Однозначний переклад (собака → der Hund)?
    │     → Обробити автоматично
    │
    ├── Кілька DE слів з різним значенням/контекстом?
    │     → Черга: 'Яке значення?'
    │       з перекладом і контекстом для кожного варіанту
    │       Дозволити вибір кількох: [a, c] або [усі]
    │
    └── Абстрактне або широке значення (відчуття, стан, іти)?
          → Черга: 'В якому контексті використовується це слово?'
            Запропонувати 3-4 кандидати з поясненням
```

**Приклад для "іти":**
```
⚠️ "іти" — яке значення руху?
  [a] gehen (ging, ist gegangen) — іти пішки
  [b] fahren (fährt, fuhr, ist gefahren) — їхати транспортом
  [c] laufen (läuft, lief, ist gelaufen) — бігти / іти (розм.)
  [d] усі три (три окремі картки)
```

**Приклад для "відчуття":**
```
⚠️ "відчуття" — яке значення?
  [a] das Gefühl, -(e)s, -e — емоційне відчуття, почуття
  [b] das Empfinden, -s, - — суб'єктивне сприйняття
  [c] die Empfindung, -, -en — фізичне відчуття (тепло, біль)
  [d] усі три
```

## Категорія 7 — Фраза чи окремі слова?

```
Багатослівний вхід
    ├── Відома стала фраза (auf jeden Fall, in der Tat)?
    │     → Обробити як одну картку-фразу
    │
    ├── Відома ідіома (jdm. auf die Nerven gehen)?
    │     → Обробити як [Rdw]
    │
    ├── Граматична конструкція (nicht nur... sondern auch)?
    │     → Обробити як фразу
    │
    └── Невідома комбінація (auf dem Tisch)?
          → Черга: 'Це стала фраза для вивчення, чи окремі слова?'
            [a] Стала фраза — одна картка
            [b] Окремі слова — обробити кожне окремо
```
